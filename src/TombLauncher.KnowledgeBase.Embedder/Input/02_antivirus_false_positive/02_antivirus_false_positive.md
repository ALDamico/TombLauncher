## Why does Windows Defender/my antivirus flag custom levels as viruses?

Because of the multiple unofficial patches applied during level building, antivirus software and indows Defender **will detect some files as viruses**, or even worse, will delete them without even advising.

**But we can grant you all the detections about that are false positives**!

---

To solve this issue, add Tomb Launcher's data folder to the exception list of Windows Defender or your other antivirus software.

**Note**: Never remove any plugin and dll!
Every file is necessary. Removing plugins or .dlls can cause crucial
issues to the game, sometimes even making a level unplayable.
