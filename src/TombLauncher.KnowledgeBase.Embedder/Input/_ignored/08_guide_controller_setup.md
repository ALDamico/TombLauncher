 **# Pair and assign controller #**
Assuming you already know how to pair your preferred controller
search it if you don't know how (to keep this guide short)

After pairing you controller, do the following:
1-Open Start menu and type "Controller" in the search bar.
2- Click on Setup USB game controller
3- Advanced , then set the controller as preferred device for older programs.

**[Screenshot – Configurazione controller legacy su Windows 10]**
Pannello superiore: ricerca nel menu Start di "controller", con "Set up USB game controllers – Control Panel" evidenziato come Best Match.
Pannello inferiore: finestra *Game Controllers* con il controller rilevato (Wireless Controller, Status: OK). Una freccia indica il pulsante **Advanced...**; nella finestra *Advanced Settings* è selezionato "Wireless Controller" come Preferred device.

Skipping this step most likely will make your controller undetected by the game
You can select one controller to be used.
To switch controller on the go ( of the same type ), disconnect the previous one, connect the new controller.
Whichever controller is connected first, the system will set it as preferred controller.
If you wish to switch controller to a different type ( PS4 to Xbox) you have start from step 1.

Now run the game.
In game options, choose controls > "Joystick" and assign the controls to match your controller:

 **# Xbox 360/S/X controller #**
Playability: Not ideal.
It has many undetected buttons, but still playable.

This works with all Xbox variants, 360 and later.
This is best convenient configuration to play the game due to limited selection on the layout detected by the game.
However you can choose the best configuration that works for you!

 **# PS DS4 controller #**
Playability: Best as original controls!
The best controller to play TR, you can map almost every button on it.

This works with PS DS4 and later, as well as aftermarket/3rd party with DS4 chipset.
This works along with DS4Windows profiles and emulated virtual controllers.
This layout based on Tomb Raider's original controls, you can change it as you like!

 **# Generic - 3rd party controllers #**
Playability: Best as original controls!
Based on PS2 style controllers, cheap, wire(d)less, some are not very comfy to hold though.
Analog button (if available) can be used to switch between joystick and D-pad move which can be locked by game.
No guarantee all of them works!!

This layout based on Tomb Raider's original controls, you can change it as you like!

 **# Notes #**
- This has been tested on some Tomb Raider (1 to 5) levels.
- TR 1 isn't playable with controller, the game originally can only
detect hard buttons, no joystick or triggers. (unless patched otherwise)
- Game and controls vary on patches and level creator ( some keys function are locked or changed accordingly)
- Vibration? Don't think so (unless patched otherwise)
- Controller must be turned on/ plugged before starting the game.
- There is no need to assign (Step left / right) hold (Walk) button and press either left or right to step sideways.
- Movements is assigned by game on default, can't be changed.
- There is no pause button, you have to use P on keyboard.
- This guide was made on Windows 10, controllers on older OS may require additional driver or setup (sorry can't test this)
