# How to play custom levels - Antivirus False Positive

**by dinne**  
*Posted: 12 Jun 2021 13:39*

---

Because of the multiple unofficial patches applied during level building, antivirus software and Windows Defender **will detect some files as viruses**, or even worse, will delete them without even advising.

**But we can grant you all the detections about that are false positives**!

---

**To bypass this issue, please do the following:**

1) Create an empty folder you will use to store your downloaded TRLE levels.
2) Turn off Windows Defender and (if you have) your other active antivirus software.
3) Add the created folder to the exception list of Windows Defender or your other antivirus software.

*Extra protection:*
4) Unzip your level inside that folder and launch the tomb4.exe file.
5) Add the tomb4.exe process to the exception list of Windows Defender or your other antivirus software.

6) Close everything and turn on Windows Defender and your antivirus again.
From that point, you'll download your levels and unzip them only inside that folder.

**Enjoy!
Levels listing: <http://www.trle.net/pFind.php>**

-:- -:- -:-

Note (1): No installation is required
and you don't need the official TR4 game to run any of these levels, as
they are completely standalone games.

Note (2): To access the levels settings (resolution, fog FX, bilinear filtering etc), press **CTRL** just before the exe starts.
If this won't work, copy the tomb4.exe file and paste a shortcut of it.
Open the properties of the shortcut with right click and add " -setup"
at the end of the path. Use this shortcut to access the settings.

Note (3): Never remove any plugin and dll!
Every file is necessary. Removing plugins or .dlls can cause crucial
issues to the game, sometimes even making a level unplayable.
