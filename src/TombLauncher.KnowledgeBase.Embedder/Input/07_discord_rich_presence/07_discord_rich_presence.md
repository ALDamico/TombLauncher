# Discord Rich Presence
**by Tomb Launcher Developers**  
*Last Edited: 16 May 2026*

## What is Discord Rich Presence?
Discord Rich Presence is a feature that displays your current activity in your Discord profile. When enabled in Tomb Launcher, other Discord users can see which Tomb Raider custom level you are playing, who authored it, and how long you have been playing.

## How do I enable Discord Rich Presence?
Open the Settings page and navigate to the "External integrations" section. Toggle the "Share play sessions on Discord" switch. The setting takes effect immediately on the next game launch.

## Does Discord need to be running for this feature to work?
Yes, Discord must be running on your computer for the Rich Presence to be displayed. If Discord is not running when you launch a game, Tomb Launcher will attempt to connect silently and fail without any error message.

## What information is shown in my Discord status?
When you launch a game, your Discord status will show:
- The name of the level you are playing
- The author of the level
- The game engine (shown as the engine logo)
- A button linking to the level's download page (if the game was installed from TRLE.net, TRCustoms.org, or AspideTR.com)
- A button linking to the Tomb Launcher website

## Is this feature enabled by default?
No, Discord Rich Presence sharing is disabled by default. You must explicitly enable it in Settings.

## Will other users be able to see which level I am playing even if my Discord status is set to invisible?
No. Tomb Launcher updates your Discord Rich Presence status, but Discord's own privacy settings still apply. If your status is set to invisible or if you have disabled activity status in Discord settings, nothing will be visible to other users.
